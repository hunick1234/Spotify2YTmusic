# Spotify2YTmusic

Spotify2YTmusic 是一個用於將 Spotify 播放清單、專輯和藝術家轉移到 YouTube Music 的應用程式。

## 功能

- 支援 Spotify , YouTube Music 等平台。
- 轉移來源平臺播放清單、專輯和藝術家到目標平臺。
- 使用者認證和授權。
